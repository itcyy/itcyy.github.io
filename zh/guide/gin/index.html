# 初识Gin

# 什么是Gin

Gin是一个用Go语言编写的web框架。它是一个类似于martini但拥有更好性能的API框架, 由于使用了httprouter，速度提高了近40倍

Go世界里最流行的Web框架，Github (opens new window)上有32K+star。 基于httprouter (opens new window)开发的Web框架。 中文文档 (opens new window)齐全，简单易用的轻量级框架。

# 安装Gin

如何安装Gin？我们安装Gin等go语言框架时，我们都会使用到安装命令，类似于node的npm、yarn，在go语言里面，我们会使用go get命令安装我们所需要的第三方库/框架，安装gin框架的命令如下：

go get -u github.com/gin-gonic/gin

如果上面命令无法安装，可以尝试一下命令。

go get -f github.com/gin-gonic/gin

如果以上命令都无法解决，那么重装go语言的编译环境。可以参考文章：Go安装|阿诺斯学习基地 (opens new window)

# Gin的简单程序

package main

import "github.com/gin-gonic/gin"

func main() {
	// 1. 创建路由
	g := gin.Default()
	// 2.绑定路由规则，执行的函数
	// gin.Context，封装了request和response
	g.GET("/", func(c *gin.Context) {
		c.String(200, "Hello Gin")
	})

	// 3.监听端口，默认在8080
	// Run("里面不指定端口号默认为8080")
	g.Run(":9000")
}

然后在浏览器输入http://localhost:端口号，就可以查看我们的运行结果了。

然后我们试试用html获取数据。

package main

import "github.com/gin-gonic/gin"

func main() {
	// 1. 创建路由
	g := gin.Default()
	// 2.绑定路由规则，执行的函数
	// gin.Context，封装了request和response
	//3.加载文件
	g.LoadHTMLGlob("templates/index.html")
	g.GET("/", func(c *gin.Context) {
		c.HTML(200, "index.html", gin.H{
			"title": "Hello Gin",
		})
	})

	// 3.监听端口，默认在8080
	// Run("里面不指定端口号默认为8080")
	g.Run(":9000")
}
